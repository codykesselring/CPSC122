Task #1 Recursive Count Digits 
Write a recursive function, int countDigits(string); that counts all the digits in a string. 
Examples:
countDigits("C6H12O6"); returns 4
countDigits("abc123"); returns 3


Task #2 Recursive Sum Digits
Write a recursive function, int sumDigits(int); that returns the sum of the digits of an integer value.
Examples:
sumDigits(12); returns 3
sumDigits(765); returns 18
sumDigits(16978); returns 31


Task #3 Recursive Palindrome 
Write a recursive function, bool isPalindrome(string); that accepts a string and determines if the string is a palindrome. A palindrome is a sequence of symbols that may be interpreted the same forward and backward. The function returns true for a palindrome, false otherwise. 
Examples:
isPalindrome("LEVEL"); returns true
isPalindrome("gonzaga"); returns false


Task #4 Recursive Linked List Max Value 
write a member function of the LinkedList class, int getMaxValue(); that returns the largest value stored in the list. Return -1 if the list is empty.
  
Task #5 Recursive Substring Generator 
Write a recursive function, void generateSubstrings(string, vector<string>&); that generates all substrings of a given string. For example, the substrings of the string “bird” are the 11 strings:
“bird”, “bir”, “bi”, “b” ,”ird”, “ir”, “i”, “rd”, “r”, “d”, “”
